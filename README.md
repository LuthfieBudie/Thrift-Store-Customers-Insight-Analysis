# Thrift-Store-Customers-Insight-Analysis
Analyzing customer behavior and store performance metrics for a thrift retail business. Using synthetic data generated from an AI-driven retail simulation, this analysis aims to uncover actionable insights regarding customer shopping patterns, peak hours, and product preferences
🛠️ Tools: Excel: Initial data exploration, data cleaning, and data type validation, Power BI: Interactive visualization and platform trend analysis.

📊 Key Insight: 
- Financial Performance: Successfully tracked a Total Revenue of $181K across 6,652 Total Transactions, establishing an Average Order Value (AOV) of $27.24
- Core Product Drivers: Jackets and Suits/Formal categories emerged as the primary revenue contributors, as visualized through the product category distribution chart.
- Peak Hours & Staffing: Identified operational sweet spots and high-traffic shopping windows by mapping transactional time data into a custom Peak Hours Matrix (Heatmap).
- Demographic Preferences: Unearthed distinct purchasing patterns between male and female shoppers to optimize targeted marketing campaigns for specific clothing categories.

📁 Repository Structure: 
- data: Documentation of the dummy dataset used
- report: The master Power BI Desktop report file
- visualizations: Dashboard screenshots and visual reports showcasing the custom gray canvas layout and rounded-corner cards.
